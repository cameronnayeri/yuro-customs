# Yuro Customs — Static Site

Plain HTML/CSS site for [yurocustoms.com](https://yurocustoms.com). No build step, no framework, no npm.

## Deploy to GitHub Pages

1. Push this folder to a GitHub repository.
2. Go to **Settings → Pages → Source**: Deploy from a branch → `main` / `/ (root)`.
3. Site goes live at `https://<user>.github.io/<repo>/`.

## Before launch

- [ ] Confirm real contact email (replace `hello@yurocustoms.com` placeholder in all four pages).
- [ ] Add physical address and business hours.
- [ ] Drop real product and portrait photography into `assets/img/` — replace `hero.jpg`, `piece-01.jpg` … `piece-06.jpg`, `portrait.jpg`.
- [ ] Add real social URLs or remove the social section.
- [ ] Replace `assets/favicon.png` with the actual favicon.
